# Gemas

Una gema es una librería o aplicación de Ruby empaquetada. Ese paquete contiene los archivos necesarios (de código, de configuración, etc.)
y la información para ser instalada en un sistema. Cada gema tiene un nombre, p.ej. `rake`, y una
versión, como `0.5.2`
